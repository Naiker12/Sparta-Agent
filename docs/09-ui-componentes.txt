1. LAYOUT PRINCIPAL
--------------------
AppShell.tsx — Shell raíz
  ├─ TitleBar.tsx (drag region + AppMenu + tabs)
  ├─ SidebarProvider
  │   ├─ AppSidebar.tsx
  │   └─ Panel Activo (según MainView)
  ├─ StatusBar.tsx
  └─ SettingsDialog.tsx (condicional)

Paneles activos (ui.store MainView.type):
  chat      → ChatArea.tsx
  editor    → EditorPanel.tsx (Monaco)
  terminal  → TerminalPanel.tsx (xterm.js)
  agents    → AgentsPanel.tsx
  sessions  → SessionsView.tsx (lazy)
  skills    → SkillsView.tsx (lazy)
  mcp       → McpView.tsx (lazy)
  channels  → ChannelsView.tsx (lazy)
  memory    → MemoryView.tsx (lazy)
