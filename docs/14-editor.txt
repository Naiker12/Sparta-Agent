1. COMPONENTES (src/)
-----------------------------------------
EditorPanel.tsx           — Panel principal (579 líneas)
  Monaco Editor + file tree sidebar
  Abre archivos via window.fs.readFile()
  Guarda via window.fs.writeFile()
  Drag-resize para ancho del explorador
  Condiciona explorador a rootPath existente

MonacoEditor.tsx          — Wrapper @monaco-editor/react
  Tema vs-dark + Geist Mono
  Theme-aware (mapea Sparta themes → Monaco)

FileTreeSidebar.tsx       — Explorador de archivos
  Árbol recursivo desde window.fs.readDir()
  Excluye node_modules/.git/.venv/etc
  Sin proyecto: solo header minimal con botón "Abrir carpeta"

FileTreeItem.tsx          — Item individual (archivo/directorio)
  Botón trash en hover
  Usa ConfirmDeleteDialog

EditorTabs.tsx            — Tabs de archivos abiertos
  Indicador de modified, botón cerrar

EditorEmptyState.tsx      — Estado vacío central (único CTA)
  Drag & drop + botón "Abrir carpeta"
  Feedback inmediato "Abriendo proyecto…" con spinner

3. INTEGRACIÓN FILESYSTEM
---------------------------
window.fs (preload.ts):
  openFolderDialog() → diálogo nativo
  readDir(path)      → lista directorio
  readFile(path)     → lee archivo
  writeFile(path, content) → escribe archivo
  deleteFile(path)   → papelera (shell.trashItem())
  deleteFolder(path) → papelera

filesystem.ipc.ts (electron/ipc/):
  fs:openFolderDialog
  fs:readDir, fs:readFile, fs:writeFile
  fs:deleteFile, fs:deleteFolder

4. FUNCIONALIDADES
--------------------
Ctrl+S / Cmd+S → guarda archivo activo
Ctrl+B          → toggle explorador
Minimap: deshabilitado
Lazy-loaded: @monaco-editor/react (no bloquea bundle)
Al eliminar archivo: cierra tab si estaba abierto
Al cerrar proyecto: limpia rootPath sin tocar disco
