2. PROTOCOLO DE COMUNICACIÓN
-------------------------------
JSON-RPC sobre stdin/stdout:
  Request:  {"id": 1, "method": "chat.stream", "params": {...}}
  Response: {"event": "stream:token", "data": {...}}

Métodos soportados:
  chat.stream       — streaming de mensajes
  memory.index      — indexar en ChromaDB
  memory.search     — búsqueda semántica
  skills.index      — obtener índice de skills
  skills.view       — ver SKILL.md

3. ENTRY POINT (main.py)
---------------------------
Configura UTF-8 IO
Arranca StdioServer (server.py)
Emite señal "ready" cuando está listo
Bucle: read json line → dispatch → write json line

4. SERVIDOR WEB (server_web.py)
----------------------------------
FastAPI + WebSocket para modo web
Alternativa a stdio cuando se ejecuta standalone

5. LANGGRAPH GRAPH
--------------------
SpartaState (extends MessagesState):
  session_id, mode, active_skills, memory_context
  thinking_tokens, tool_calls_this_turn
  subagent_results, plan, current_step
  plan_complete, reflection_retries

Nodos: planner → agent → tools → reflection
Edges condicionales: agent→tools, agent→subagent, tools→agent, etc.
