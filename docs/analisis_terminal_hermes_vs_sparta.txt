1. ESTADO ACTUAL (después de mejoras)
═══════════════════════════════════════════════════════════════════

  HERMES AGENT tiene 3 superficies de terminal:
  ───────────────────────────────────────────
  ✅ CLI    - hermes_cli/console_engine.py (comandos curados, no shell real)
  ✅ Web    - web/ (React + xterm.js, conecta por WebSocket al gateway)
  ✅ Desktop - apps/desktop/ (Electron + node-pty + xterm.js, tabs, persistencia)

  SPARTA AGENT tiene 2 superficies:
  ──────────────────────────────────
  ✅ Web    - src/ (React + xterm.js, modo web via WebSocket)
  ✅ Desktop - electron/ (Electron + node-pty + xterm.js)

2. MEJORAS APLICADAS A SPARTA
═══════════════════════════════════════════════════════════════════

  BASADO EN LO QUE HERMES HACE:

  ╔══════════════════════════════════════════════════════════════╗
  ║  TerminalPanel.tsx - MULTIPLES TABS + PERSISTENCIA          ║
  ╚══════════════════════════════════════════════════════════════╝
  ● Cada tab es una sesion PTY independiente (como Hermes)
  ● Las sesiones NO se destruyen al ocultar el panel
  ● Tabs con indicador de conexion (verde/rojo)
  ● Boton + para nueva terminal, X para cerrar
  ● Reinicio de sesion individual por tab
  ● WebGL renderer (fallback a canvas si no disponible)
  ● Unicode11 addon (soporte Unicode completo)
  ● WebLinks addon (URLs clickeables)

  ╔══════════════════════════════════════════════════════════════╗
  ║  AppShell.tsx - TERMINAL PERSISTENTE                        ║
  ╚══════════════════════════════════════════════════════════════╝
  ● Cambio de AnimatePresence (destroy on close) a motion.div
  ● El panel se oculta visualmente pero xterm+PTY siguen vivos
  ● Animacion suave al mostrar/ocultar (height + opacity)
  ● Ya no se pierde el historial al cerrar/abrir terminal

  ╔══════════════════════════════════════════════════════════════╗
  ║  terminal.ipc.ts - AGENTE MEJORADO                          ║
  ╚══════════════════════════════════════════════════════════════╝
  ● agent-write busca cualquier terminal activa si no se da ID
  ● Nuevo IPC terminal:list-sessions para monitorear sesiones
  ● Deteccion mejorada del shell en Windows

  ╔══════════════════════════════════════════════════════════════╗
  ║  preload.ts - NUEVA API                                      ║
  ╚══════════════════════════════════════════════════════════════╝
  ● window.terminal.listSessions()
